## 背景

当前鲁迅题目提示词过于模板化，无法有效引导玩家逐步缩小答案范围：

**原始提示词：**
```
1. 行旅清晰
2. 政见成章
3. 流派留痕
4. 心志隽永
5. 家学可循
6. 师承昭著
7. 地域特色
```

**问题：**
- 7条都使用模板化句式（清晰/成章/留痕/隽永/可循/昭著/特色）
- 语义骨架雷同，维度重复
- 第7条"地域特色"太虚，无法收束
- 整体无法指向鲁迅

---

## 设计目标

遵循 Xuanheng 工作流规则：
1. **维度分散**：7条落在不同主维度
2. **强度阶梯**：从弱到强，逐步缩小范围
3. **前6条不锁词**：只提供维度指引
4. **第7条收束**：承担最终确认职责
