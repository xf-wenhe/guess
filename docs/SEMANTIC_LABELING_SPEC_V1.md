# 语义百分比分标注规范 v1

## 目的
这份规范只服务一个目标：让模型更接近真实自然语义，而不是只提高平均指标。

适用范围：
- 答案词 vs 用户猜词
- 短中文词、短短语、常见口语、提示式表达、离题词、乱输入

## 标注原则
1. 先判断自然语义关系，再给分。
2. 不按“同类”给高分；同类但语义远，只能低到中分。
3. 不按“能联想到同一个场景”给高分；必须看是否接近同一语义核心。
4. 不按“像提示词”自动给高分；提示式表达要看它离答案语义本体有多近。
5. 先想“普通中文母语者会不会觉得这两个词本来就像”，再打分。

## 分段定义
### 0-10
完全无关、乱输入、跨域错配。
例子：
- 猫咪 vs 刘备
- 星球 vs 不知道
- 听课 vs 高处直冲

### 11-25
有非常弱的联想，但自然语义上基本不相似。
例子：
- 星球 vs 数学
- 敬佩 vs 老马识途

### 26-40
同类、同领域、同场景，但语义仍明显不同。
例子：
- 月食 vs 流星雨
- 语文 vs 地质
- 许嵩 vs 邓紫棋

### 41-55
相关但不接近本义；提示感存在，但不足以视作近义。
例子：
- 豆花 vs 勺舀
- 挟天子 vs 掌控象征
- 快乐 vs 悲伤
- 高兴 vs 难过

反义词统一按 50% 左右标注：它们共享语义维度，但方向相反，不再作为 0-25 的硬负例。

### 56-70
中等相关，普通人会觉得有明显联系，但不会直接当成同义。
例子：
- 刮骨疗毒 vs 华佗
- 望远镜 vs 天文器

### 71-85
近义、常见别称、稳定替代表达。
例子：
- 医生 vs 大夫
- 诸葛亮 vs 孔明
- 孙悟空 vs 齐天大圣

### 86-95
高度同义、昵称、本词的强别名。
例子：
- 猫咪 vs 猫

### 96-100
完全一致或仅形式变化，不允许用于泛化判断，只能做边界锚点。
例子：
- 汉堡 vs 汉堡
- 跳跃 vs 跳跃

## 必填错误类型
只允许使用以下 error_type：
- exact
- alias
- near_synonym
- same_category_but_far
- antonym_mid
- hint_like
- cross_domain_negative
- nonsense
- collocation_not_equivalent
- abstract_confusion

## 必填备注要求
每条标注必须写一句 why_wrong 或 rationale，回答下面至少一个问题：
1. 为什么当前模型分数不对？
2. 这两个词真正接近在哪？
3. 它们为什么虽然同类，但不该高分？

## 标注禁忌
1. 不允许只因为“同类”就打 60+。
2. 不允许只因为“像线索”就打 70+。
3. 不允许把 exact/self-match 混进 holdout 里拉高指标。
4. 不允许同一批样本同时用于训练、校准、最终评测。

## 推荐工作流
1. 把线上坏例先写进失败样本回收表。
2. 每周集中审核一次 corrected_score。
3. 审核通过后再合入 anchor/train。
4. 冻结一份 holdout，不参与训练和校准。
